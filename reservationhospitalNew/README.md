# 🏥 Graha Medika - Hospital Reservation System

A modern web-based hospital appointment reservation system and electronic medical records (EMR) portal, designed with an **Elite Medical Minimalist** aesthetic (Ultra-Clean, aesthetic, and highly professional Siloam Hospitals style).

---

## 🌟 Key Features

* 📅 **Dynamic Calendar (Flatpickr)**: Booking calendar dates automatically disable (dim out) on the doctor's off-days, dynamically fetched from the database via AJAX based on the selected doctor's schedule.
* 👥 **Multi-Dashboard**: Intuitive separate portals for Patients (Appointment Booking & Medical Records) and Doctors (Manage Appointments & Patient History).
* 🛡️ **Premium UI/UX**: Modern, ultra-clean, and fully responsive interface utilizing Outfit & Inter typography with a sophisticated Navy-Teal color palette.

---

## 🛠️ Tech Stack & Libraries

* **Backend**: PHP & MySQL (MySQLi)
* **Frontend**: HTML5, CSS3, Vanilla JS
* **Libraries**: Flatpickr (Aesthetic Airbnb Calendar Theme) & SweetAlert2 (Elegant Modal Alerts)

---

## 🚀 Quick Setup & Installation

1. **Move Folder**: Copy the `reservationhospital` project folder into your local server directory (`C:\xampp\htdocs\`).
2. **Create Database**: Open **[http://localhost/phpmyadmin/](http://localhost/phpmyadmin/)** and create a new database named `reservation_hospital`.
3. **Import SQL File**: Select the `reservation_hospital` database, click the **Import** tab, choose the `reservation_hospital.sql` file in the project folder, and click **Import**.
4. **All Set**: Open your browser and navigate to **[http://localhost/reservationhospital/](http://localhost/reservationhospital/)** 🎉
